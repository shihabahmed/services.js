Service.Popup Show alert message. 
Service.Popup.show({
	titlebar: true, // whether to show the title bar
	title: 'Message', // Title of the box (when the title bar is visible)
	message: 'This is the message content...', // message to show
	timeout: 0 // if value > 0, alert box will automatically disappear after given seconds

});

Service.Confirm Show confirmation dialog. 
Service.Confirm.show({
	titlebar: true, // whether to show the title bar
	title: 'Message', // Title of the box (when the title bar is visible)
	message: 'This is the message content...', //message to show
	yes: function () { }, // callback function to call when clicked "Yes"
	no: function () { } // callback function to call when clicked "No"
});

Service.Prompt Show prompt dialog. 
Service.Prompt.show({
	title: 'Message', // Title of the box
	height: 80, // Height of the textarea
	placeholder: 'Write your message here...', // Placeholder text
	onSubmit: function(message) {
		// do something if clicked 'Submit'
    }
});

Service.PasswordChecker To add strength indicator for a password field.
<input type="password" class="password-field" />

Service.PasswordChecker.init('.password-field');

Service.Loader To show loader on a button/div, add class loader, then call show().
<button class="btnSubmit" data-loader>Submit</button>
<input type="button" value="Submit" class="btnSubmit" data-loader />
<div class="contentLoader" data-loader></div>

Service.Loader.show('.btnSubmit');
Service.Loader.show('.contentLoader');
To hide a specific loader, call hide() as below. 
Service.Loader.hide('.btnSubmit');
Service.Loader.hide('.contentLoader');

Service.ScrollTo Scrolls to a specific element in the page. 
Service.ScrollTo('h3', [optionalMarginForScrollTo]);

margin is an integer which, if not set or set to 0, the element scrolled to, will be touching the top edge of the browser window.


Service.WindowScroll If you need to run some javascript code on window.scroll event, put those code in a function. 
function init() {
	// your js code...
}

function anotherFunction(a) {
	// do something with 'a'...
}
and then.. 
Service.WindowScroll.add(init);
Service.WindowScroll.add(anotherFunction, a);
At a certain point if you don't want a function to run every time the window is scrolled.. 
Service.WindowScroll.remove(anotherFunction);

Service.WindowLoad Works the same way as Service.WindowScroll but for window.load event. 
Service.WindowLoad.add(init);
Service.WindowLoad.add(anotherFunction, a);
Service.WindowLoad.remove(anotherFunction);

Service.WindowResize Works the same way as Service.WindowScroll but for window.resize event. 
Service.WindowResize.add(init);
Service.WindowResize.add(anotherFunction, a);
Service.WindowResize.remove(anotherFunction);

Service.DomReady Works the same way as Service.WindowScroll but for document.ready event. 
Service.DomReady.add(init);
Service.DomReady.add(anotherFunction, a);
Service.DomReady.remove(anotherFunction);
